# 🧡 Feliz Día del Padre

Una página web hecha con cariño como regalo del Día del Padre — una experiencia
interactiva de una sola página, con animaciones, galería de momentos, una carta
y un "abrazo virtual".

## 🌅 Cómo verla

- **En tu computador:** abre `index.html` en cualquier navegador moderno.
- **En línea (GitHub Pages):** en el repositorio ve a
  **Settings → Pages → Build and deployment → Branch: `main` / `/root`** y guarda.
  En un par de minutos quedará publicada en
  `https://<tu-usuario>.github.io/<nombre-del-repo>/`.

## 📁 Estructura

```
index.html      → el contenido y la estructura
styles.css      → el diseño y las animaciones CSS
script.js       → la interactividad (partículas, lightbox, contadores, abrazo…)
Imagenes/       → las fotos
```

## ✏️ Personalizar

- La **firma de la carta** está marcada en `index.html` con un comentario
  `✏️ PERSONALIZA AQUÍ` — cambia `tu hijo` por tu nombre.
- Los textos de cada sección están en español, en `index.html`, listos para editar.

## ✨ Detalles técnicos

- HTML, CSS y JavaScript puros — sin dependencias ni build.
- Diseño responsive (se adapta a móvil).
- Respeta `prefers-reduced-motion` para accesibilidad.
